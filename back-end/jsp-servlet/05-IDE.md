## IDE

### 의의

- 개발의 여러 단계를 통합한 환경을 제공

  > 1. 코드 작성 / 수정
  > 2. 컴파일
  > 3. 배포
  > 4. 톰캣 서버 재시작
  > 5. 브라우저 요청

  - 위 5단계를, 한 번의 Run으로 통합

- Eclipse

  - Java EE 버전
  - Tip
    - tools 폴더: 사용하는 개발 툴을 모아놓고 관리
    - workspace 폴더: 각 프로젝트마다, 작업 공간 폴더를 생성하여 관리
  - Project 생성 / 환경 설정
    - New Dynamic Web Project
    - Target runtime 
      - Apache Tomcat v9.0: 해당 폴더 "...\apache-tomcat-9.0.xx" 선택
