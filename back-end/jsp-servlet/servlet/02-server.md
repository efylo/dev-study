## Server

### Web server

- HTTP 요청을 직접적으로 받는 위치
- 요청을 동적으로 처리하기 위해 Server app을 호출
  - HTTP url 및 본문 데이터 참조

---

### WAS(Web Application Server)

- Web server로부터 파싱된 데이터를 받는다. 
- 데이터에 상응하는 비즈니스 로직을 수행한다. 
- 수행 후 결과물을 다시 web server로 전송한다.
