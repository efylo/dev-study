## Redirection

1. 사용자의 Http Request를 받아, 로직을 수행
2. 로직 수행 이후에 사용자에게 보여주고 싶은 페이지가 존재한다면 Redirection을 사용
   - Http Response 객체에, redirect 요청을 포함하여 전송
   - response.sendRedirect(url);
